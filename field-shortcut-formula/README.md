# 字段捷径公式版配置

这里放的是字段捷径开发助手要用的 JSON：

```text
default-value-customization.json
```

## 配置内容

当前只有一个字段捷径：默认值自定义。

字段配置面板里只有一个选择项：来源字段。支持文本、数字、日期、单选、多选、复选框和查找字段。

公式模板很简单：

```text
[{$ sourceField.name $}]
```

也就是直接引用用户选中的来源字段。

## 输出类型

字段捷径公式版要求在 `presetFieldConfig.fieldProperty.dataType` 里写一个输出类型。它不能根据来源字段动态变化。

所以当前配置先用文本类型承载结果。用户需要数字、日期、选项等类型时，在飞书多维表格里改字段类型。

## 怎么调试

不用启动本地服务。

1. 打开字段捷径开发助手。
2. 粘贴 `default-value-customization.json`。
3. 点击“添加字段”。
4. 选择来源字段。
5. 查看字段结果。

调试助手创建的字段只对使用调试助手的用户可见。要给其他人使用，需要走字段捷径插件发布流程。
