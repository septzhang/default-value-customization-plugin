# 默认值自定义 PRD

## 产品定位

默认值自定义是飞书多维表格字段捷径插件。第一版不做侧边栏，只为新建字段填充来自同表普通字段或查找字段的默认值。

## 开发指南校准

参考字段捷径公式版开发指南后，第一版正式提交物调整为公式版字段捷径 JSON，路径为：

```text
field-shortcut-formula/default-value-customization.json
```

公式版字段捷径通过公式字段实现默认值引用。它会随来源字段变化实时计算，不是一次性写入普通字段值。因此，本文后续关于“手动刷新”“自动新增记录填充”“选项自动创建”的内容只适用于普通前端插件或后续服务端/工作流方案，不属于公式版字段捷径第一版提交范围。

## 第一版范围

- 只支持新建字段捷径，不支持把已有字段转换成该类型。
- 公式版输出为公式字段，随来源字段实时计算。
- 不提供手动刷新历史记录。
- 不提供一次性写入普通字段值。
- 不提供自动创建单选/多选选项。
- 不承诺后台批量写入自动化。

## 字段支持

目标字段类型默认与来源字段相同。用户需要改类型时，通过飞书多维表格原生字段类型能力完成，插件不提供目标类型选择。

来源字段支持同表普通字段和同表查找字段。查找字段只可作为来源，不可作为目标。跨表取值由用户先创建飞书原生查找字段。

插件不做类型转换，只写入目标字段能直接接受的值。

## 手动刷新

手动刷新入口在字段配置面板，按钮文案为“刷新历史记录”。第一版只支持整张表，不提供预览页面。

刷新策略包括：

- 仅填充空值，默认。
- 覆盖已有值。

来源为空始终跳过，不清空目标字段。目标值和来源值相同时不重复写入。

## 自动新增记录填充

默认开启，可关闭。新记录出现后短延迟检查，来源为空时有限重试。自动模式下不自动创建单选/多选选项，只记录跳过原因。

## 选项创建

手动刷新时，单选/多选目标遇到不存在的选项，执行前只提示将新增的选项数量并要求确认，不展示预览列表。自动模式不创建选项。

## 风险控制

大表刷新显示分级提醒。超过 50,000 条记录仍允许执行，但必须二次确认，并提示耗时、中断和部分完成风险。

执行支持停止后续批次，不支持回滚、撤销、续跑或日志导出。
